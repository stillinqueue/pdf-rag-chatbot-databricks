# Sample Questions for PDF Q&A Chatbot

## Purpose

This document contains sample questions for testing the PDF Q&A chatbot.

The goal is to check whether the chatbot can:

* Answer questions from the uploaded PDF
* Retrieve relevant document chunks
* Show source chunks
* Reject unrelated questions
* Avoid answering from outside knowledge

The sample PDF used in this project is:

```text
Introduction_to_Tableau.pdf
```

---

## How to Use These Questions

Use these questions inside the final chatbot notebook:

```text
05_pdf_qa_with_free_llm
```

Example:

```python
print(ask_pdf_with_llm("What is Tableau used for?"))
```

Or use the interactive chatbot loop and type the questions manually.

## Expected Response for In-Scope Questions

For questions related to the PDF, the chatbot should return an answer based on the retrieved context.

Example question:

```text
What is Tableau used for?
```

Expected response style:

```text
Answer:
Tableau is used for data visualization and business intelligence. It helps users connect to data, explore information, and create visual dashboards.

Sources:
- Introduction_to_Tableau.pdf, chunk_id=2, score=0.8421
- Introduction_to_Tableau.pdf, chunk_id=5, score=0.9174
```

The exact wording may vary because the answer is generated by the free Hugging Face model.

---

## Expected Response for Out-of-Scope Questions

For questions unrelated to the PDF, the chatbot should not answer from general knowledge.

Example question:

```text
What is the capital of Japan?
```

Expected response:

```text
I could not find a confident answer in the uploaded PDF. Please ask a question related to the document.
```

This confirms that the chatbot is using the uploaded PDF as its knowledge source.

## Retrieval Testing Checklist

Use this checklist while testing the chatbot.

| Test Area                        | Expected Result                         |
| -------------------------------- | --------------------------------------- |
| Basic Tableau question           | Retrieves relevant chunks               |
| Visualization question           | Retrieves Tableau visualization content |
| Dimensions and measures question | Retrieves relevant concept chunks       |
| Dashboard question               | Retrieves dashboard-related chunks      |
| Unrelated geography question     | Rejected by guardrail                   |
| Unrelated programming question   | Rejected by guardrail                   |
| Personal question                | Rejected by guardrail                   |
| Sources displayed                | Source file, chunk ID, and score shown  |
| Answer grounded in PDF           | No unsupported external answer          |

## Notes

The relevance threshold may need small adjustments depending on the PDF, chunk size, and embedding scores.

Current threshold used in the project:

```python
RELEVANCE_THRESHOLD = 1.3
```

If valid PDF questions are rejected, increase the threshold slightly.

If unrelated questions are accepted, decrease the threshold slightly.
